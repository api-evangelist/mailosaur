# Mailosaur

Mailosaur is an email and SMS testing platform for developers and QA teams. It provides a REST API for creating test inboxes, capturing and retrieving messages, running assertions, performing deliverability analysis (spam scoring, SPF/DKIM/DMARC), and integrating email and SMS testing into CI/CD pipelines. The platform also supports authenticator (TOTP/2FA) testing.
